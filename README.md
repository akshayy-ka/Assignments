# Task Manager

A simple Task Manager web application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Add tasks using the Enter key
- Responsive and modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Files

- day1.html
- day1.css
- day1.js

## How to Run

1. Download the project files.
2. Open day1.html in any web browser.
